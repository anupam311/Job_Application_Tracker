# Job Application Tracker

This project is a simple Python-based Job Application Tracker that
allows users to:

-   Add new job applications\
-   Update application statuses\
-   View all stored applications\
-   Search applications by company name\
-   Exit the system safely

## Features

### 1. Submit New Application

Users can enter: - Company Name\
- Position Title\
- Date Applied\
- Current Status

All entries are stored in a list as dictionaries.

### 2. Modify Application Status

Users can update the status of any previously saved application by
providing the company name.

### 3. Show All Applications

Displays all saved job entries in a readable format.

### 4. Look Up Application by Company

Searches the list for a matching company name and shows its record.

### 5. Quit

Ends the program safely.

------------------------------------------------------------------------

## How to Run

1.  Ensure you have Python 3 installed.

2.  Save the program file as `application_tracker.py`.

3.  Run the script using:

    ``` bash
    python application_tracker.py
    ```
